Welcome to Attorney Online!

Attorney Online is a fan-made game for roleplaying and doing Ace Attorney-style cases in an off-the-cuff format.

There are a variety of active servers in AO with their own communities. Please be sure to follow their rules, and, if applicable, download their base packs, which may be found in each server's description.

You will be downloading the Vanilla base pack, which is the classic set of characters that has not changed since the old 1.7.5 days. This is a very minimal set (~250 MB); however, some servers may require content packs of up to 7 GB in size. If disk space is a concern, we invite you to use webAO to join these particular servers.
